#  Análisis de Eficiencia de Algoritmos en Python

Este proyecto mide el tiempo de ejecución de distintos algoritmos para comparar su eficiencia según su complejidad temporal.

---

##  Objetivo

Analizar cómo crece el tiempo de ejecución de algoritmos con diferentes complejidades:

- **O(1)** → Constante  
- **O(n)** → Lineal  
- **O(n²)** → Cuadrática
